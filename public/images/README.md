# Logo Assets

Place your logo images in this directory:

- `logo.png` — Dark mode logo (purple/dark cat)
- `logo_white.png` — Light mode logo (blue/light cat)

Both images will be automatically used by the website:
- **Navbar** (top-left logo / Easter Egg trigger)
- **Footer** (bottom-right, next to copyright)
- **Favicon** (browser tab icon)

If the images are missing, a text fallback "TD" will be shown.
